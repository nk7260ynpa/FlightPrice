# FlightPrice 專案概述

## 目標

每日自動追蹤特定日期的機票價格，記錄價格變化趨勢，並透過視覺化介面呈現票價隨時間的變動，協助使用者掌握最佳購票時機。

## 核心功能

1. **機票價格追蹤** — 每日擷取指定日期、航線的機票價格並儲存至資料庫
2. **資料儲存** — 使用資料庫持久化儲存歷史票價資料
3. **視覺化呈現** — 提供圖表展示票價隨時間的變化趨勢

## 技術棧

- **語言**：Python
- **執行環境**：Docker container
- **資料庫**：待定
- **視覺化**：待定
- **排程**：每日定時執行價格擷取任務
