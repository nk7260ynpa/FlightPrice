## MODIFIED Requirements

### Requirement: 新增追蹤班機

系統 SHALL 提供表單讓使用者僅需輸入班次代碼與出發日期，系統自動查詢航空公司、出發地、抵達地並填入。若查到多段航班，SHALL 顯示航段選擇下拉選單。

#### Scenario: 輸入班次代碼後自動帶入資訊（單段）
- **WHEN** 使用者輸入班次代碼並觸發查詢，且結果為單段航班
- **THEN** 系統自動填入航空公司、出發地、抵達地欄位

#### Scenario: 輸入班次代碼後顯示航段選擇（多段）
- **WHEN** 使用者輸入班次代碼並觸發查詢，且結果為多段航班
- **THEN** 系統顯示航段選擇下拉選單（如「SIN → TPE」「TPE → NRT」），使用者選擇後自動填入對應欄位

#### Scenario: 查詢失敗時允許手動輸入
- **WHEN** 航班查詢 API 無法找到該班次
- **THEN** 系統顯示提示訊息，開放航空公司、出發地、抵達地欄位供手動輸入

#### Scenario: 成功新增班機
- **WHEN** 使用者填寫班次代碼與出發日期並提交表單
- **THEN** 系統在 tracked_flights 建立紀錄，頁面重新載入並顯示更新後的清單

#### Scenario: 必填欄位未填
- **WHEN** 使用者未填寫班次代碼或出發日期即提交表單
- **THEN** 頁面顯示欄位驗證錯誤提示
